dabl.preprocessing.clean¶

dabl.preprocessing.clean(X, type_hints=None, return_types=False, verbose=0)[source]¶

Public clean interface

Parameters:
type_hints : dict or None

If dict, provide type information for columns. Keys are column names, values are types as provided by detect_types.

return_types : bool, default=False

Whether to return the inferred types

verbose : int, default=0

Verbosity control.
